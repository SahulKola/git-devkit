### Accessibility expectations

- Target: WCAG AA baseline.
- Required behaviors:
  - Visible focus ring on all interactive controls.
  - Keyboard-operable navigation and submenu access.
  - Skip link to main content.
  - Adequate text contrast for both light and dark themes.
