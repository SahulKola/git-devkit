# 🎯 To Get Started Right Now

## git-devkit Overview

**git-devkit** is a comprehensive developer toolkit for Git advanced users.

**git-multi-ssh** is the core tool within git-devkit for managing multiple Git SSH identities.

## All Fixed ✅

The regex error is fixed and setup now has a **single entry point**.

## One Command (Works Everywhere)

```bash
npm run setup
```

That's it! This:
- ✅ Detects your OS (Windows/macOS/Linux)  
- ✅ Checks prerequisites
- ✅ Installs dependencies
- ✅ Links the **git-multi-ssh** CLI globally
- ✅ Shows next steps

## Then Run

```bash
git-multi-ssh
```

Follow the prompts to create your first Git identity. Done! 🎉
