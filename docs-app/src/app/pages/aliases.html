Installation Guide

Install the complete alias set globally on macOS, Windows, or Linux.

@if (isRecommended('powershell')) {

🪟 Windows 
irm https://raw.githubusercontent.com/sahulkola/git-devkit/main/setup-aliases.ps1 | iex

🍎 macOS · 🐧 Linux 
curl -fsSL https://raw.githubusercontent.com/sahulkola/git-devkit/main/setup-aliases.sh | bash

setup-aliases.bat
