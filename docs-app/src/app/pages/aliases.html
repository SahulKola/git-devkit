Installation Guide

Install the complete alias set globally so every repository gets the same shortcuts.

One-line install

curl -fsSL https://raw.githubusercontent.com/sahulkola/git-multi-ssh/main/setup-aliases.sh | bash

Manual install

git clone https://github.com/sahulkola/git-multi-ssh.git
cd git-multi-ssh
bash setup-aliases.sh

Verify

git aliases                   # list all configured aliases
git s                         # quick status
git lg                        # graph log
